🌍 TiliTrip — Планировщик путешествий и бюджета
TiliTrip — это десктопное приложение на Python для тех, кто любит держать свои поездки под контролем. Оно позволяет не только планировать маршруты по дням, но и следить за расходами в реальном времени, превращая хаос подготовки в четкий план.


✨ Основные функции
🗺️ Управление поездками
Создание и поиск: Добавляйте новые путешествия с указанием города вылета и даты. Быстрый поиск поможет найти нужный тур в большом списке.
Гибкие статусы: Отслеживайте состояние поездки (Планируется, В процессе, Завершено). Цветовая индикация помогает визуально отделить текущие дела от архивных.

💰 Планирование и Бюджет
Подневный план: Добавляйте локации и активности для каждого дня путешествия.
Контроль расходов: Указывайте стоимость для каждого пункта. Приложение автоматически суммирует общие затраты на поездку.
Чек-лист: Отмечайте выполненные пункты плана двойным кликом.
⚙️ Технологические особенности

Локальное хранение: Все данные хранятся в базе данных SQLite (файл tilitrip.db) — интернет не нужен для доступа к планам.
Звуковое сопровождение: Приятные уведомления при добавлении задач и успешных расчетах.
🛠 Стек технологий
GUI: customtkinter (современный интерфейс в стиле Material Design).
Backend: Python 3.x.
Database: SQLite3.
Date Management: tkcalendar.
